# Smart Food Pre-Order System

## Project Overview
The Smart Food Pre-Order System is a web-based application developed using Django that allows users to order food online before arriving at the restaurant. The system helps reduce waiting time and improves order management for both customers and administrators.

## Features
- User registration and login system
- Browse available food menu
- Add items to cart
- Place food orders online
- Admin panel to manage menu and orders

## System Workflow
1. User logs into the system.
2. User browses the food menu.
3. Items can be added to the cart.
4. User places an order.
5. Admin can view and manage incoming orders.

## Project Structure
The project includes the following components:

- `food/` – Django application handling food ordering logic
- `templates/` – HTML templates for user interface
- `static/` – CSS and static files
- `manage.py` – Django project management file
- `db.sqlite3` – Database storing user and order information

## Technologies Used
- Python
- Django
- SQLite
- HTML
- CSS

## How to Run the Project

1. Install Python and Django.
2. Clone the repository.
3. Navigate to the project directory.
4. Run the server using:

```
python manage.py runserver
```

5. Open the browser and access the local server.
